# OpenCV Webcam Manipulation
We use OpenCV in C++ to manipulate the webcam feed and write it to a virtual
video device.

This is developed in Void Linux.

## Usage

### Key bindings
| Key        | Function |
|:-------------:|:-------------:|
| `Esc`      |  Quit |
| `Space`    |  Toggle frame processing |
| `r` | Change capture resolution |
